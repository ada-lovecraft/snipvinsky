st2snippets
===========

My collection of Sublime Text 2 Snippets

To Use (OS X)
---
	$ cd /Users/[your_username>]/Library/Application Support/Sublime Text 2/Packages
	$ mkdir codevinksy
	$ git clone https://github.com/codevinsky/st2snippets.git codevinsky
